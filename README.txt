This application is an Android blackbox application for have-not-so-much-money
users. Features include
- Continous recording
- Event recording
- Sending SOS message to configured phone number(s) when an accident detected
- Playing recorded event and video
- Displaying a trajectory on a map while driving

Due to the characteristics of this application, it may consume lots of power.
You need to connect power source while using this application.

Please, report the problem when you encounter one at https://github.com/ivehicle/AnBox/issues
